
On first run the instructor asks a few quick questions — including **the one real
{{running_example}}** you want to work on. From then on **every exercise runs on
your example**, and it becomes your **capstone**.

Then, any time:
- **"Let's do Session X"** — start a specific session
- **"Continue my course"** — resume where you left off
- **"Jump to Session X"** — jump around when flexible mode is on
- **"Use sequential mode" / "Use flexible mode"** — choose guided order or flexible navigation
- **"Show my progress"** — see your checkmark view
- **"Show my time"** — see active learning time by total and session
- **"Pause timer" / "Resume timer"** — exclude breaks or side quests from active time
- **"What's next?"** — get the next session
- **"I'm stuck on X"** — debug
- **"Define X"** — get a plain-language definition
- **"Quiz me"** — practice weak/current concepts
